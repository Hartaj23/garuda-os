# templates

Engineering Document Templates v1.0

Version: 1.0

Last Updated: 2026-07-06

## Purpose

Contains reusable engineering document templates for Project Garuda missions.

Use these templates during mission specification, planning, approval, implementation review, completion, and release preparation. Copy a template into the appropriate `docs/` location when producing a permanent record.

## Owner

Project Garuda Engineering Governance

## Template Sequence

```
Mission Specification
↓
Implementation Plan
↓
Approval
↓
Implementation
↓
Architecture Review
↓
Completion Report
↓
Release Notes / Sprint Closure
```
