# Changelog

All notable changes to Project Garuda will be documented in this file.

The project follows semantic versioning as required by the repository architecture.

## v0.3.0-alpha - 2026-07-03

### Added

- Universal Memory Framework for platform memory objects, memory types, confidence, metadata and
  Platform Core inheritance.
- Memory Source & Provenance Framework for source descriptors, acquisition metadata, provenance
  references and traceability.
- Memory serialization and validation integration with deterministic memory payloads and Platform
  Core validation compatibility.
- Memory Index Contract for descriptive index field, descriptor and contract metadata.
- Memory Retrieval Contract for descriptive request, criteria, response, status and metadata
  models with opaque memory identifiers.
- In-Memory Reference Store for process-local `UniversalMemory` references with exact identifier
  add, get, remove, enumerate, clear, statistics and validation behavior.
- Memory Foundation platform integration certification suite and permanent sprint certification
  record.
- Memory Foundation SDK documentation covering developer usage, architecture, API reference, best
  practices, extensions and examples.
- Sprint closure report and release notes for `v0.3.0-alpha`.

## v0.2.0-alpha - 2026-07-02

### Added

- Universal Object Framework for platform object identity, metadata, lifecycle state, audit fields,
  validation hooks, behavior registration and canonical object inheritance.
- Universal Object Registry for canonical object registration, lookup, enumeration and duplicate
  protection.
- Serialization Framework for deterministic `GarudaObject` dictionary and JSON payloads.
- Universal Relationship Framework for deterministic relationship payloads between immutable object
  IDs.
- Lifecycle Event Framework for deterministic lifecycle event payloads.
- Validation Framework for result aggregation, validation errors, severities, categories and object
  validation helpers.
- Platform integration certification suite and permanent Platform Core certification record.
- Platform Core SDK documentation covering developer usage, architecture, API reference, best
  practices, extensions and examples.
- Sprint closure report and release notes for `v0.2.0-alpha`.

## v0.1.0-alpha - 2026-07-02

### Added

- Repository foundation for GAR-SPRINT-0001 Task 1.
- Governance files, monorepo skeleton, directory documentation and repository validation tests.
