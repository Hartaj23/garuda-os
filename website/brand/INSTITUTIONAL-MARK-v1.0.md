## Adoption

The Founding Generation hereby adopts **C-03 Baseline** as the official institutional mark of Project Garuda.

Adopted: 2026-07-08

---

## Form

**C-03 Baseline** — a horizontal rule with centered vertical tick.

The line on which institutional text rests.

| Property | Value |
| --- | --- |
| Type | Geometric — text baseline |
| ViewBox | 0 0 24 24 |

## Institutional Principle

Project Garuda's identity resides in its constitutions, governance, architecture, engineering, preservation, and public philosophy.

The institutional mark exists to acknowledge that identity.

It does not define it.

Should a future generation determine that a different mark better serves the institution, that decision shall be undertaken through the same disciplined governance by which this mark has been adopted.

The institution preserves legitimacy above permanence.
