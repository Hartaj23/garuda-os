# Reasoning Foundation SDK Documentation

## Purpose

This documentation set describes the completed GAR-SPRINT-0006 Universal Reasoning Foundation SDK.
It covers the service-independent reasoning object, descriptive input and provenance models,
strategy and chain contracts, runtime workspace, validation integration, serialization
compatibility, and platform certification implemented in `packages.reasoning`.

## Platform Boundary

The SDK documentation covers only implemented Reasoning Foundation behavior. It does not describe
reasoning engines, strategy execution, chain execution, inference, conclusion generation, planning,
decision making, orchestration, persistence, search, AI integration, REST APIs, frontend behavior,
or workflow functionality as implemented.
