# Execution Foundation SDK Documentation

## Purpose

This documentation set describes the completed GAR-SPRINT-0009 Universal Execution Foundation SDK. It
covers the service-independent Execution object, descriptive input and provenance models, strategy and
chain contracts, runtime workspace, validation integration, serialization compatibility, and
platform certification implemented in `packages.execution`.

## Platform Boundary

The SDK documentation covers only implemented Execution Foundation behavior. It does not describe
execution behavior, strategy execution, chain execution, outcome computation, reference resolution,
provenance evaluation, scheduling, orchestration, optimization, persistence, search, AI integration,
REST APIs, frontend behavior, autonomous behavior, or workflow functionality as implemented.
