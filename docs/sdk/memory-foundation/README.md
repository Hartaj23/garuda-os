# Memory Foundation SDK Documentation

## Purpose

This documentation set describes the completed GAR-SPRINT-0003 Memory Foundation SDK. It covers
the service-independent memory models, contracts, and runtime reference container implemented in
`packages.memory`.

## Platform Boundary

The SDK documentation covers only implemented Memory Foundation behavior. It does not describe or
imply implemented Knowledge, Context, Reasoning, Decision, Agent, persistence, database, retrieval
engine, search, REST, frontend, workflow, trading, or portfolio functionality.
