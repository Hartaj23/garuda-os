## Purpose

This documentation set describes the completed GAR-SPRINT-0010 Interface Foundation SDK. It covers the
certified Interface substrate, canonical contracts, lifecycle and boundary model, translation
framework, validation framework, registry catalog, Platform Core inheritance, and constitutional
certification implemented in `packages.interface`.

## Platform Boundary

The SDK documentation covers only implemented Interface Foundation behavior. It does not describe
runtime execution, provider integration, transport registration, service location, dependency
injection, plugin discovery, persistence, authentication, authorization, REST APIs, MCP integration,
OpenAI integration, orchestration, scheduling, or outbound translation as implemented capability.
