# Decision Foundation SDK Documentation

## Purpose

This documentation set describes the completed GAR-SPRINT-0007 Universal Decision Foundation SDK.
It covers the service-independent Decision object, descriptive input and provenance models, strategy
and chain contracts, runtime workspace, validation integration, serialization compatibility, and
platform certification implemented in `packages.decision`.

## Platform Boundary

The SDK documentation covers only implemented Decision Foundation behavior. It does not describe
decision execution, strategy execution, chain execution, outcome computation, reference resolution,
provenance evaluation, orchestration, planning, optimization, persistence, search, AI integration,
REST APIs, frontend behavior, autonomous behavior, or workflow functionality as implemented.
