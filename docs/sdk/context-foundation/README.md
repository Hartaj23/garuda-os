# Context Foundation SDK Documentation

## Purpose

This documentation set describes the completed GAR-SPRINT-0005 Context Foundation SDK. It covers
the service-independent context models, descriptive contracts, validation integration, runtime
workspace, and platform certification implemented in `packages.context`.

- Universal Context Framework

- Context Serialization and Validation Integration
- Context Composition Contract
- Context Selection Contract
- Context Workspace
- Context Platform Integration and Quality Certification

## Platform Boundary

The SDK documentation covers only implemented Context Foundation behavior. It does not describe
context engines, composition engines, selection engines, retrieval engines, search engines,
reasoning, inference, persistence, database, AI, REST, frontend, workflow, trading, or portfolio
functionality as implemented.
