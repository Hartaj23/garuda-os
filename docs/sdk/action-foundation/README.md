# Action Foundation SDK Documentation

## Purpose

This documentation set describes the completed GAR-SPRINT-0008 Universal Action Foundation SDK. It
covers the service-independent Action object, descriptive input and provenance models, strategy and
chain contracts, runtime workspace, validation integration, serialization compatibility, and
platform certification implemented in `packages.action`.

- Action Platform Integration and Quality Certification

## Platform Boundary

The SDK documentation covers only implemented Action Foundation behavior. It does not describe
action execution, strategy execution, chain execution, outcome computation, reference resolution,
provenance evaluation, scheduling, orchestration, optimization, persistence, search, AI integration,
REST APIs, frontend behavior, autonomous behavior, or workflow functionality as implemented.
