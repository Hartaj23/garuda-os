This documentation set describes the completed GAR-SPRINT-0002 Platform Core SDK. It covers the
service-independent Universal Object System implemented in `packages.objects`.

## Platform Boundary

The SDK documentation covers only implemented Platform Core behavior. It does not describe or imply
Knowledge Graph, Memory, AI, Workflow Engine, persistence, REST, database, trading, or portfolio
functionality.
