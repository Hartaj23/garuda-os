# docs/adr

## Purpose

Contains Architecture Decision Records (ADRs) for Project Garuda.

Approved ADRs are the normative architectural baseline. Gate-authorized drafts may appear here while
under Founder review; draft status is explicit in the ADR header and does not confer approval.

## Authority

ADRs derive authority exclusively from GAR Constitutions. They never establish constitutional scope.
They never authorize engineering unless a separate Founder gate grants sprint authority.
