## Founder Decision

Design Tokens v1.0 is hereby authorized.

Authorization is limited to translating the approved Visual Philosophy v1.0 and Design Language v1.0 into reusable visual tokens for the Project Garuda public website and related public communication artifacts.

---

## Permitted

This authorization permits definition of:

- typography system,
- color palette,
- spacing scale,
- layout grid,
- diagram styling rules,
- accessibility requirements.

---

## Not Permitted

This authorization does not permit:

- logo design,
- website implementation,
- public deployment,
- social launch,
- external communication campaign.

---

## Institutional Guidance

Design Tokens v1.0 shall serve as visual implementation guidance only.

The website introduces.

The repository governs.

---

End of Design Tokens v1.0 Authorization
