# docs/governance

## Purpose

Contains institutional governance artifacts that describe how Project Garuda is governed between
engineering sprints. These documents record institutional direction and lifecycle models. They do not
authorize engineering work.

## Authority

Governance documents in this directory are **descriptive only** unless explicitly ratified as
constitutional law elsewhere. They never supersede GAR constitutions, ADRs, or approved sprint
specifications.
