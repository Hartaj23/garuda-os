## The Adoption Print Exercise

Print the [Canonical Page Specification v1.0](../../website/brand/CANONICAL-PAGE-SPECIFICATION-v1.0.md) **twice**.

### Copy A — Without the mark

The page as specified. Structure alone. No institutional mark.

### Copy B — With C-03 Baseline

The same page, with [C-03 Baseline](../../website/brand/mark-selection/candidates/c-03-baseline.svg) placed exactly where the specification proposes — site header, 24px, left of navigation, per [INSTITUTIONAL-MARK-EXPLORATION-v1.0.md](../../website/brand/INSTITUTIONAL-MARK-EXPLORATION-v1.0.md) placement rules.

Both copies:

- A4
- Black and white
- No color

Then ask only one question:

> **"Does the page become more itself — or merely more branded?"**

That is the only question that matters.

| Answer | Action |
| --- | --- |
| C-03 makes the page feel **more like itself** | Adoption has been earned |
| C-03 merely makes the page look **"finished"** | Wait |

## Institutional Readiness

An institution is ready to adopt its mark when removing the mark no longer removes its identity.

The [Stranger's Report](PRINT-EXERCISE-INDEPENDENT-READER-OBSERVATION-MISSION-001-PHASE-B.md) demonstrated exactly that.

C-03 Baseline is no longer being asked to **create** Project Garuda.

It is simply being invited to **accompany** it.

That is the highest compliment an institutional mark can receive.
