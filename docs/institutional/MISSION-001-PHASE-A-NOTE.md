---

Mission 001 Phase A established the first permanent separation between Project Garuda's institutional repository and its public communication layer, preserving repository authority while enabling future public engagement.

The `website/` directory was created at repository root — distinct from `docs/`, `packages/`, and `apps/` — so that public communication would never be confused with constitutional, governance, architectural, or engineering authority.

This architectural decision was deliberate rather than incidental.

The website introduces the institution.

The repository remains the institution.
