# docs/institutional

## Purpose

Contains institutional literature — narrative and reference documents that explain Project Garuda's
governance philosophy and history.

These documents are **informational only**. They do not create constitutional, architectural,
sprint, or engineering authority.

## Authority

Institutional literature is subordinate to GAR constitutions, ADRs, approved sprint specifications,
and committed repository state.
